# 安装 Flutter SDK

首先我们要安装 Flutter SDK，参考 [获取 Flutter SDK](/2019/02/07/how-to-building-flutter-environment/)

# 安装相关构建工具

使用下面的命令安装 webdev

```bash
flutter pub global activate webdev
```

添加 C:\\flutter\\.pub-cache\\bin（替换为你的实际安装地址）到环境变量
