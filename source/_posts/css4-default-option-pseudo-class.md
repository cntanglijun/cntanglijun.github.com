# 默认项伪类

> 这个伪类也是 CSS3UI 规范的一部分。

这个伪类叫做默认项伪类，它在一组其它相同元素中选择默认元素。它可以选择下拉列表/菜单或者上下文菜单的默认项。

# 语法

```css
:default {
  /* declarations */
}
```

# 示例

```css
:default {
  background-color: gray;
}
```

这个例子设置下拉列表/按钮组的默认项的 `background-color` 为 `gray`。
