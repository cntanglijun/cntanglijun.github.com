RequireJS 是一个 JavaScript 文件和模块加载器。它既可以在浏览器环境中使用也可以在其他 JavaScript 环境中使用，例如 Rhino 和 Node。使用模块化脚本加载器如 RequireJS 将提升你们代码的执行速度和质量。

RequireJS 兼容 **IE6+**，**Firefox 2+**，**Safari 3.2+**，**Chrome 3+**，**Opera 10+** 。
